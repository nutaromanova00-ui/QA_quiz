# 🐞 QA Тренажёр — Подготовка к собеседованию по ручному тестированию

**Интерактивный тренажёр, который превращает подготовку к собеседованию в увлекательный процесс**

🎯 **80 вопросов** | 🎴 **Карточки + Тесты** | 💾 **Прогресс сохраняется**

## 📌 О проекте

**QA Тренажёр** — это веб-приложение для подготовки к собеседованию на позицию **Junior / Trainee QA Engineer**. 

Вместо скучного заучивания теории ты получаешь **два режима обучения**:

| Режим | Что даёт |
|-------|----------|
| 🃏 **Карточки** | Быстрое повторение — смотришь вопрос, думаешь, переворачиваешь для проверки |
| ✍️ **Тест** | Проверка знаний с пояснениями — при ошибке получаешь развёрнутый ответ |

### 🎯 Почему это работает?

- **Активное вспоминание** — самый эффективный метод запоминания
- **Контекстные пояснения** — ты не просто запоминаешь ответ, а понимаешь тему
- **Прогресс сохраняется** — видишь свой рост и возвращаешься к тренировкам

## ✨ Ключевые особенности

| | |
|---|---|
| 🎴 **80 реальных вопросов** | Никакой воды — только то, что действительно спрашивают на собеседованиях |
| 🔄 **Умные неправильные ответы** | Дистракторы звучат правдоподобно, тест не становится "угадайкой" |
| 💾 **Прогресс в localStorage** | Статистика правильных ответов не сбрасывается при перезагрузке |
| 📊 **Визуальная строка прогресса** | Ты видишь, как растёт твой уровень подготовки |
| 🎲 **Перемешивание вопросов** | Запоминаешь суть, а не последовательность ответов |
| 📱 **Адаптивный дизайн** | Удобно заниматься с телефона, планшета или ноутбука |

## 🚀 Быстрый старт

1. Скачай репозиторий как ZIP или клонируй его
2. Распакуй архив (если скачивал ZIP)
3. Открой файл `index.html` в любом современном браузере

> Никаких зависимостей, никакой установки — открыл и работаешь ✅

## 🎮 Как пользоваться

**Режим 1: Карточки**

1. Читаешь вопрос
2. Думаешь над ответом
3. Нажимаешь на карточку — она переворачивается
4. Сравниваешь свой ответ с правильным
5. Кнопки Назад / Вперёд — листаешь коллекцию
6. Перемешать — меняет порядок карточек

**Режим 2: Тест**

1. Выбираешь вариант ответа из 4
2. Получаешь мгновенную обратную связь: Верно — краткое подтверждение; Ошибка — правильный ответ + развёрнутое пояснение
3. Следующий вопрос — двигаешься дальше
4. Перемешать — меняет порядок и генерирует новые варианты

**Отслеживание прогресса**

- Вверху экрана — твоя статистика (например, ✅ 15 / 30)
- Прогресс-бар показывает процент правильных ответов
- Данные сохраняются автоматически в браузере

## 🔧 Кастомизация

**Хочешь добавить свои вопросы?**

Открой файл `data.js` и добавь в массив:

{ 
  question: "Твой вопрос?", 
  answer: "Правильный ответ (кратко и чётко)" 
}

Вопрос автоматически появится и в карточках, и в тесте ✅
